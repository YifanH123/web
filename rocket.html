Rocket Avionics Bay

A custom-engineered sensor suite for high-altitude data acquisition and real-time flight monitoring.

Project Overview

Capturing data at Mach speeds requires robust hardware. This project focused on designing a compact avionics sled capable of recording flight dynamics while withstanding high G-forces and vibration.

Technical Implementation

Sensor Fusion: The core system integrates a LSM6DS3TR-C IMU for orientation tracking and a BMP390 barometric pressure sensor for precise altitude determination. Data is logged at 100Hz to an SD card via an Adalogger Feather M0.

Telemetry: A 915MHz RFM95W radio module transmits critical packets (altitude, velocity, GPS coordinates) to a ground station in real-time, aiding in recovery.

System Specs

MCU Feather M0 Adalogger
Telemetry 915MHz LoRa
Sampling Rate 100Hz
Max Altitude 3,350 ft
Power 3.7V LiPo 500mAh
