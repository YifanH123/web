Multi-Band Audio Equalizer

A precision analog circuit for shaping audio frequencies using active filters and MOSFET amplification.

Project Overview

This project focused on the design and implementation of a modular audio equalizer. The goal was to create a device capable of boosting or cutting specific frequency bands (Bass, Treble) without introducing significant noise or harmonic distortion.

Technical Implementation

Active Filters: I utilized operational amplifiers (TL072) in a Sallen-Key topology to create 2nd-order Low-Pass and High-Pass filters. This allows for independent control over distinct frequency ranges.

Amplification Stage: The output stage features a Class AB amplifier design using complementary MOSFET pairs (IRF530/IRF9530). This configuration was chosen to balance power efficiency with linear signal reproduction, minimizing crossover distortion.

Technical Specs

Frequency Bands 2-Band (Low, High)
Op-Amps TL072 (Low Noise JFET)
Output Power 15W per Channel
THD < 0.1% @ 1kHz
Cutoff Frequencies 200Hz / 2.5kHz
